# Nest Saga - Orchestrator Service

# Getting Started

## Server Requirements

- Node.js 20.12.1

## Installing preparation

1. Default Application $BASEPATH : `/home/app.user/saga-orchestrator`

2. Copy the .env file from .env.example under $BASEPATH

# Build with Docker

## 1. Setup docker

```bash
  make setup
```

```bash
  make up
```

## 3. Run dev mode

```bash
  make dev
```
